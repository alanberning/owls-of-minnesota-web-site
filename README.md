# owls-of-minnesota-web-site

Capstone project using HTML and CSS for class project. This project focuses to inform the viewer
about the owl species found in the state of Minnesota. The pages themselves contain very clean and
minimal design. The focus on this project was to create a universal look using a CSS stylesheet.

Resources were used from the website https://avibirds.com/owls-in-minnesota/

The most challenging thing for me was the dropdown menubar. This element was not included in
the curriculum until the following semester, and though simple, it took a lot of trial and error
to get it to work properly for me.
